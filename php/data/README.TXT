PHP Generic Registration Module (PHP-GEREMO)
============================================

This directory contains the (dynamic) data that are generated and used by the
PHP Generic Registration Module.

Should you need to place this directory elsewhere, please update the 'data_dir'
configuration parameter accordingly (see the sample 'config.php' for further
information).

ATTENTION: This directory MUST be writable (and readable) by PHP!

CRITICAL: THIS DIRECTORY MUST NOT BE ACCESSIBLE FROM THE WEB!!!
